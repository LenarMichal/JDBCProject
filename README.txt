Aplikacja JAVA FX za pomocą programu SQL Developer oraz bazy danych Oracle Database 11g Express Edition tworzy tabele w bazie danych które następnie pobiera do interfejsu graficznego JAVA. Dodatkowo istnieje możliwość zapisu do bazy danych nowej tabeli zawierającej wybrane przez użytkownika dane. Jest to symulacja elektronicznego dziennika szkolnego.
